Classic Game
Basic game is with 10*20 random numbers between 1-9, with 120 seconds time limit. You can play this game by dragging and selecting the numbers which is
selected in the area of rectangle created by dragging. Once you select the numbers that are sum of 10, the numbers eliminated.
If you clear the board, the game is over and the score will be recorded as time.
If you did not clear the board, the game is still over if the time is over, and the score will be recorded as the number you eliminated.
During the game you can pause the game; You could restart the whole game, resume the current game, close the whole app, or go back to the home screen.
You can check the whole records in the Leaderboard - Classic menu on home screen.

Customized Game

You can customize the number of rows, columns, time limit, and the sum of numbers to be eliminated. The game is with rows*columns random numbers
between 1 to (sum of numbers - 1). Everything else is same as classic game.
Please refer to the Customized_rule.png for the detail.
You can check the whole records in the Leaderboard - Customized menu on home screen.

PVP:
You and your opponent sharing the board and the user will battle with the opponent from anywhere once both are all online.
The score will be given to the who eliminate the numbers, and if one eliminate more, that user will win.
*is currently unavailable since it must use internet service with sharing
